# 🚀 Инструкция по деплою bloom.html на сервер

## **Шаг 1: Открой SSH консоль Timeweb**

1. Зайди на Timeweb в панель управления
2. Открой SSH консоль

---

## **Шаг 2: Скопируй содержимое bloom.html**

1. Открой файл `E:\Island\_temp\bloom.html` на компьютере
2. Скопируй ВСЁ содержимое (Ctrl+A, Ctrl+C)

---

## **Шаг 3: Создай файл на сервере**

В SSH консоли выполни:

```bash
cd /root/api_server/api_server/public
nano bloom.html
```

---

## **Шаг 4: Вставь содержимое**

1. В редакторе `nano` вставь скопированный HTML (правой кнопкой мыши → Paste)
2. Сохрани файл: `Ctrl+O`, затем `Enter`
3. Выйди из редактора: `Ctrl+X`

---

## **Шаг 5: Проверь создание файла**

```bash
ls -lh bloom.html
```

Должен увидеть что-то вроде:
```
-rw-r--r-- 1 root root 15K Oct 16 23:45 bloom.html
```

---

## **Шаг 6: Перезапусти Caddy (если нужно)**

```bash
docker restart api_server-caddy-1
```

---

## **Шаг 7: Проверь в браузере**

Открой в браузере:

```
http://marabot.tw1.ru/bloom.html
```
