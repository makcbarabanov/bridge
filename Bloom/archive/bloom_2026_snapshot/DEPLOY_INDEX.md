# 🚀 Инструкция по деплою обновленной страницы Bloom на сервер

## ⚠️ Важно
На сервере уже есть `bloom.html` со старой версией. Заменим её обновленной версией из `index.html`.

## **Способ 1: Через SSH консоль Timeweb (Рекомендуется)**

### **Шаг 1: Открой SSH консоль Timeweb**

1. Зайди на Timeweb в панель управления
2. Открой SSH консоль

---

### **Шаг 2: Скопируй содержимое index.html**

1. Открой файл `E:\BLOOM 2026\index.html` на компьютере
2. Скопируй ВСЁ содержимое (Ctrl+A, Ctrl+C)

---

### **Шаг 3: Замени файл bloom.html на сервере**

В SSH консоли выполни:

```bash
cd /home/www/public
nano bloom.html
```

---

### **Шаг 4: Вставь содержимое**

1. В редакторе `nano` вставь скопированный HTML (правой кнопкой мыши → Paste)
2. Сохрани файл: `Ctrl+O`, затем `Enter`
3. Выйди из редактора: `Ctrl+X`

---

### **Шаг 5: Проверь создание файла**

```bash
ls -lh bloom.html
```

Должен увидеть что-то вроде:
```
-rw-r--r-- 1 root root 19K Nov 17 00:25 bloom.html
```

---

### **Шаг 6: Перезапусти Caddy (если нужно)**

```bash
docker restart api_server-caddy-1
```

---

### **Шаг 7: Проверь в браузере**

Открой в браузере:

```
https://marabot.tw1.ru/bloom.html
```

---

## **Способ 2: Через scp (если настроен SSH ключ)**

Выполни в терминале:

```bash
scp "E:\BLOOM 2026\index.html" root@marabot.tw1.ru:/home/www/public/bloom.html
```

Затем перезапусти Caddy:
```bash
ssh root@marabot.tw1.ru "docker restart api_server-caddy-1"
```
