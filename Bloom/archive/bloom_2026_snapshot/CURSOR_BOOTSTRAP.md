# Старт сессии в Cursor / AI Studio (не для Telegram-бота)

Этот файл **не** подмешивается в ответы Telegram-бота. Используй в IDE, когда открываешь репозиторий `bridge` и работаешь с кодом или каноном Блума.

Корень канона по умолчанию: `Bloom/extracted/BLOOM 2026/` (или путь из `BLOOM_HOME`).

## Восстановление контекста в новой сессии

1. Прочитай `README.md` в этой папке.
2. При необходимости: `DOCUMENTATION/journey.md`.
3. При необходимости: `DIALOG_BLOOM_FORGE_letters-with-bloom.txt` (архив переписки).
4. Системная инструкция для **бота** задаётся в `BLOOM_START_PROMPT.txt` + `Bloom/bloom_context.py` — не дублируй старые пути `E:\`.
