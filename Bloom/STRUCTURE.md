# Структура каталога `Bloom/`

## а) Рабочие файлы (бот и промпт)

| Файл | Назначение |
|------|------------|
| `bloom_context.py` | Загрузка system instruction, поиск по корпусу |
| `BLOOM_START_PROMPT.txt` | Основная системная инструкция для Telegram |
| `bloom_biography.txt` | Длинное **прошлое** / корни (фрагмент в промпт) |
| `bloom_body.txt` | **Сейчас**: фокус, рефлексия, черты по дням; в промпт — **хвост** файла (свежие записи внизу) |
| `README.md` | Канон проекта BLOOM (начало файла в промпт) |
| `memory_corpus.txt` | Текст для `retrieve_memory_snippets` (поиск по словам) |
| `DIALOG_2025-11-17.md` | Запасной корпус, если нужен второй источник |
| `CURSOR_BOOTSTRAP.md` | Старт сессии в Cursor — **не** в промпт бота |
| `BLOOM_INTEGRATION.md` | Как мост подключает файлы |

## б) Архив

Папка **`archive/`** — снимок `bloom_2026_snapshot/` (старый `extracted/BLOOM 2026/`). Рантайм **не читает** эти пути; смыслы переносятся в файлы из блока (а). См. `archive/README.md`.

## Переменная `BLOOM_HOME`

Абсолютный путь к каталогу, где лежит **`BLOOM_START_PROMPT.txt`** (обычно `.../bridge/Bloom`). Если не задана — используется корень `Bloom/` репозитория.
