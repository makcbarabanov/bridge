# Inbox — раздаточный бокс

Сюда клади **временные** материалы: скриншоты, выгрузки, черновики справок.

**Правило:** когда суть перенесена в `user_profiles/…/`, `dialogy/` или другую основную структуру — **удали файлы из inbox**, папку можно оставить пустой.

Не храни здесь единственную копию важных данных.
